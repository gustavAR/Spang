To run the demo of this sprint, please begin with starting the PC program (/Spang-PC/Spang-PC_C-sharp.exe). Subsequently, start the android app (installed by running "Spang-mobile.apk" on an android device). Choose to scan QR-code to obtain the correct IP-adress and port by scanning the code appering on the computer screen. Click connect on the phone.

No larger changes has been made to the functionallity compared to sprint 3, instead refactorings has been made (see system architecture documentation)
