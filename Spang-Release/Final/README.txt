To run the demo of this sprint, please begin with starting the PC program (/Spang-PC/Spang-PC_C-sharp.exe). 
Subsequently, start the android app (installed by running "Spang-mobile.apk" on an android device). 
Choose to scan QR-code to obtain the correct IP-adress and port by scanning the code appering on the computer screen. 
Click connect on the phone.

In this final release, we have added the functionality to setup and use shortcuts!
