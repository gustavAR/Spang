To run the demo of this sprint, please begin with starting the PC program (/Spang-PC/Spang-PC_C-sharp.exe). Subsequently, start the android app (installed by running "Spang-mobile.apk" on an android device) and input the IP-adress corresponding to the computer running the PC-program. Press the "connect" button on the android device.

In order to move the cursor, use the mobile touchscreen as a touchpad (scroll by using two fingers).
To change the computer volume, use the volume control buttons on the android device.

Pressing the menu button and choosing "settings" will bring up the settings view.
